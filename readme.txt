〇操作について

　・PS4コントローラでの操作を想定してチュートリアルの記載やキー割り当てを行っています
　　遊ぶ際は是非ともコントローラでお願いします

　・ゲームを実行するより先にコントローラを接続していないとコントローラが認識されません

　・下記の操作はゲーム起動時にもイラスト付きで確認できます


〇PS4コントローラ
　
　・メニュー操作

　　〇ボタン　　　　　　　＝決定
　　✕ボタン　　　　　　　＝キャンセル
　　方向キー・スティック　＝移動
　　OPTION　　　　　　　　＝メニュー呼び出し

　・ステージ内操作

　　〇ボタン　　　　　　　＝ブロック設置
　　✕ボタン　　　　　　　＝女の子の誘導
    R1ボタン　　　　　　　＝ブロック射出
　　方向キー・スティック　＝移動
　　OPTION　　　　　　　　＝メニュー呼び出し

〇キーボード

　・メニュー操作

　　ENTER                 ＝決定
　　WASD                  ＝移動
　　ESC　　　　　　　　　 ＝メニュー呼び出し

　・ステージ内操作

　　ENTER　　　　　　　　 ＝ブロック設置
　　SPACE                 ＝女の子の誘導
　　右SHIFT               ＝ブロック射出
　　WASD                  ＝移動
　　ESC                   ＝メニュー呼び出し
